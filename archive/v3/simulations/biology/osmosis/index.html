Procedure
  1. Cut 6 equal potato cylinders using a cork borer
  2. Weigh each cylinder and record initial mass
  3. Place one cylinder in each sucrose solution
  4. Leave for 30 minutes (simulated)
  5. Remove, blot dry, and reweigh each cylinder
  6. Calculate % change and plot graph
