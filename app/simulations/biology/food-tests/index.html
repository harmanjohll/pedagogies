Procedure Guide
  1. Select — Choose a food sample to test.
  2. Iodine test — Transfer sample to a tube. Add iodine solution. Blue-black = starch present.
  3. Benedict's test — Transfer sample to a tube. Add Benedict's solution. Heat in the water bath. Brick-red = reducing sugar.
  4. Biuret test — Transfer sample to a tube. Add NaOH(aq), then add CuSO₄(aq). Purple = protein.
  5. Emulsion test — Transfer sample to a tube. Add ethanol, shake, then add water. White cloudy = lipid/fat.
  6. Conclude — Identify which macronutrients are present.
